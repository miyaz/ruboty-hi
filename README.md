# Ruboty::Hi

ruboty handler to say "hi" to everyone.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-hi'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruboty-hi

## Usage

    @ruboty hi - hi
